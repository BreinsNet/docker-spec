# Docker::Spec


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'docker-spec'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install docker-spec

## Usage

Add this line to the spec_helper.rb

```
require 'docker-spec'
```

And run rspec to build your docker containers
